About This Project

This platform creates dynamic psychological profiles of users based on their text and social media data. The personas evolve over time as more data is collected, providing increasingly accurate insights.

Key Features:

  • AI-powered personality trait analysis
  • Sentiment evaluation of text content
  • Dynamic profiles that evolve over time
  • Behavioral pattern identification
  • Social media content analysis

Hackathon Demo: This prototype demonstrates the concept with real AI analysis when an OpenAI API key is provided, or with synthetic data if not.
